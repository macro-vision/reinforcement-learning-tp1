## Installation

1. Installer une version récente de Python (^3.11), puis installer poetry
```bash
python --version # 3.11
pip install -U pip poetry
```

2. Installer les dépendances
```
poetry install
```

3. Lancer le serveur jupyter
```
poetry shell
jupyter-lab
```

## Tâches
